SNAKE GAME

This project recreates the classic Snake Game using Python’s Turtle library, combining simplicity with engaging gameplay. The player controls the snake to collect food, growing longer with each bite while avoiding collisions with the walls or itself. The game includes smooth movement mechanics, dynamic score tracking, and a game-over screen to enhance the experience. Built entirely with the Turtle module, this project demonstrates fundamental concepts like event handling, collision detection, and object-oriented programming in Python.
